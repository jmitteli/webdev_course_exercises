# Weekly Web Development Exercises 📚

A repository for my weekly programming exercises as part of a web development course. Each week focuses on a different topic, building on previous skills.

## 🗂️ Course Topics

The exercises change weekly, covering the following topics throughout the course:

- HTML
- CSS
- Responsive Design
- Pre-compiled CSS (e.g. Sass/SCSS)
- JavaScript
- Object-Oriented Programming with JavaScript
- HTTP Requests & Open Data APIs

## 🛠️ Technologies Used

- HTML5
- CSS3
- Sass / SCSS
- JavaScript (ES6+)

## 🚀 How to Run

Most exercises are plain HTML/CSS/JS files and can be opened directly in a browser.
No build tools are required unless stated in the specific week's folder.

## 📖 About

These exercises are part of a web development course and are intended for learning purposes.
Each folder contains the weekly assignments and any related assets.
